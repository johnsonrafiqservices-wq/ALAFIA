                  3 SIMPLE STEPS TO SMS:
═══════════════════════════════════════════════════════════════════

STEP 1: Sign Up (FREE - 2 minutes)
───────────────────────────────────

   Go to: https://africastalking.com/
   Click: "Get Started"
   Sign up with your email
   
   ✓ FREE sandbox with test credits
   ✓ No credit card required
   ✓ Unlimited testing


STEP 2: Get API Key (2 minutes)
────────────────────────────────

   After signing up:
   1. Log in to Africa's Talking
   2. Go to "Sandbox"
   3. Click "Settings" → "API Key"
   4. Click "Generate API Key"
   5. Copy the API key (looks like: atsk_abc123...)


STEP 3: Add to .env File (1 minute)
────────────────────────────────────

   1. Open (or create) .env file in project root
   2. Add these 3 lines:

   SMS_PROVIDER=africas_talking
   AFRICAS_TALKING_API_KEY=paste_your_key_here
   AFRICAS_TALKING_USERNAME=sandbox


═══════════════════════════════════════════════════════════════════
                    THAT'S IT! TEST IT:
═══════════════════════════════════════════════════════════════════

   Run: python test_sms.py

   This will:
   ✓ Show your configuration
   ✓ Let you send a test SMS
   ✓ Verify everything works


═══════════════════════════════════════════════════════════════════
                   ENABLE IN SYSTEM:
═══════════════════════════════════════════════════════════════════

   1. Go to: http://localhost:8000/admin/appointments/remindersettings/
   
   2. Check ✓ "SMS Enabled"
   
   3. Click "Save"
   
   4. Done! SMS reminders will now be sent automatically
