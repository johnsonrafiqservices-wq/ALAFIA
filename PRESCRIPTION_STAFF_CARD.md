# 📋 Prescription Print & Email - Staff Quick Card

## 🖨️ HOW TO PRINT A PRESCRIPTION

### Method 1: From Prescription List (Recommended)
```
1. Pharmacy → Prescriptions
2. Find prescription in list
3. Click 🖨️ (printer icon)
4. Click "Print" button or press Ctrl+P
5. Verify 1 page in preview
6. Print!
```

### Method 2: Direct URL
```
/pharmacy/prescriptions/<ID>/print/
Example: /pharmacy/prescriptions/5/print/
```

### Auto-Print
```
Add ?auto_print=true to URL
Example: /pharmacy/prescriptions/5/print/?auto_print=true
```

---

## 📧 HOW TO EMAIL A PRESCRIPTION

### From Prescription List
```
1. Pharmacy → Prescriptions
2. Find prescription in list
3. Click 📧 (envelope icon)
4. Confirm in popup dialog
5. Success! Patient receives email
```

**Note**: Email button is disabled if patient has no email address.

## ✅ WHAT'S ON THE PRESCRIPTION

### Essential Information
- ✅ Clinic name, address, phone, email
- ✅ Prescription ID (RX-00XXX)
- ✅ Prescription date
- ✅ Patient: Name, ID, Age, Gender, Phone
- ✅ Prescriber: Doctor name, date
- ✅ Medication: Name, dosage, frequency, duration, quantity
- ✅ Special instructions (if any)
- ✅ Dispensed info (if applicable)
- ✅ Legal terms
- ✅ Signature line

### Format
- **Layout**: Compact tables
- **Pages**: 1 page (A4)
- **Colors**: Blue & Green (Alafia theme)
- **Style**: Professional medical document

## ⚡ QUICK TIPS

### For Best Results
✅ **Print**: Enable "Background graphics" in print settings  
✅ **Paper**: Use A4 or Letter size  
✅ **Preview**: Always check print preview shows 1 page  
✅ **Email**: Verify patient email before sending  
✅ **Status**: Prescription shows "Dispensed" badge if dispensed  

### Common Issues
❌ **Email button disabled?** → Patient has no email address  
❌ **Print shows 2 pages?** → Check browser zoom (should be 100%)  
❌ **Missing info?** → Update patient/prescription data first  
❌ **Colors not showing?** → Enable background graphics in print settings

## 🔍 TROUBLESHOOTING

| Problem | Solution |
|---------|----------|
| Print button not working | Check browser pop-up blocker |
| Email not sending | Verify email server configuration |
| Wrong clinic info | Update clinic settings in admin |
| Layout broken | Clear browser cache (Ctrl+Shift+R) |
| Patient email missing | Add email to patient profile |
